同期設定

標準は GAS 同期です。ローカルモードは必要なときだけ切り替えます。

標準同期では自動でこのURLを使います。

標準同期では、他の端末でもURL入力なしでそのまま使えます。ローカルモードは端末ごとの下書き用です。

公開手順

GitHub Pages と GAS をつなぐ最短ルートです。

`index.html` を GitHub リポジトリのルートに置く。
GitHub に push して、Repository Settings の Pages を有効化する。
Build and deployment は `Deploy from a branch`、Branch は `main`、Folder は `/ (root)` にする。
GAS を Web App として公開する。標準同期URLは画面に組み込み済みです。
LINE Bot 側は GAS の Webhook を維持しつつ、必要ならこの画面の保存ロジックを同じシートに寄せる。

操作履歴

最近の更新や同期を残します。

もし次に進めるなら、GAS 側に `GET` と `POST` の受け口を追加して、この画面から直接在庫を更新できる形まで整えられます。
